# README – Organización Jornada de Fútbol Sala

Este proyecto es una aplicación web interactiva en HTML, CSS y JavaScript diseñada para organizar y analizar una jornada o torneo de Fútbol Sala.

FUNCIONALIDADES PRINCIPALES:
- Gestión de grupos (2A, 2B, I2A, I2B)
- Creación de equipos y plantillas de jugadores
- Generación automática de calendarios
- Registro de resultados, puntos extra y análisis
- Clasificaciones automáticas con medallas y equipo menos goleado
- Gráficos de puntos por equipo (Chart.js)
- Exportación a Excel y guardado local
- Modo oscuro y navegación por pestañas

TECNOLOGÍAS:
- HTML5, CSS3, JavaScript
- Chart.js
- XLSX.js

Para usar la aplicación, abre el archivo HTML en tu navegador y sigue los pasos de configuración dentro de cada grupo.
